# 📝 Todoist

A native iOS task management app that helps users organize their daily tasks across different states (Todo, Doing, Done) — built with Objective-C using the MVC architectural pattern.

---

## 📱 Features

- **Task Management** — Add, edit, and delete tasks with ease
- **Task States** — Track progress with three distinct states: Todo, Doing, and Done
- **Prioritization** — Set task priorities (Low, Medium, High) to focus on what matters most
- **Reminders & Notifications** — Schedule local notifications to remind you of upcoming tasks
- **Attachments** — Add attached files to your tasks
- **Local Storage** — Saves all tasks locally using `NSUserDefaults` and `NSKeyedArchiver` for full offline access

## 🏗 Architecture & Tech Stack

- **Language:** Objective-C
- **Architecture:** MVC (Model-View-Controller)
- **UI Framework:** UIKit (Storyboards)
- **Persistence:** NSUserDefaults with `NSKeyedArchiver` (Secure Coding)
- **Local Notifications:** UserNotifications framework
